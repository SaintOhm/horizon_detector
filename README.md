Порядок действий должен быть таким:
1.  Обрезаем изображение до квадрата.
2.  Уменьшаем разрешение, чтобы меньше элементов было.
3.  Переводим в оттенки серого.
4.  Находим контуры. Самый большой должен быть небом по идее.
5.  Горизонт точно не на краях изображения, так что эти точки убираем.
    Можно использовать для того, чтобы понять, где верх и низ.
7.  Находим границы.
    Потому что горизонт на границе неба и земли.
8.  Фильтр точек контуров по границам.
9.  Подгоняем точки горизонта под np.polyfit.
    Отсюда получаем крен с поправкой на верх/низ.
10. Строим нормаль от центра изображения к горизонту.
    Отсюда получаем тангаж через FOV камеры
 
На деле вся загвоздка в пункте 4: оно часто захватывает всякие поля или водоёмы. 
В конце функции я пытался найти подходящее решение, переставляя этапы
-   Объединить рядом стоящие пиксели в один по np.max, чтобы мелкие блики убрать
-   Блюр по различным диапазонам и порогам
